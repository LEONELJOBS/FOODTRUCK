PAGINA DE CONTROL DE VENTAS

Como usar:
1. Abre el archivo index.html en Google Chrome, Edge o cualquier navegador moderno.
2. Ingresa precio y cantidad vendida de cada producto.
3. Ingresa lo que llego por QR y el monto adicional para sumar con QR.
4. Presiona "Guardar en historial" para guardar el cierre del dia.

Nota importante:
- La pagina detecta automaticamente el dia de trabajo.
- Si haces el cierre antes de la hora limite configurada, guarda el registro con fecha del dia anterior.
- El historial se guarda en el navegador usando localStorage. Si cambias de navegador o borras datos del navegador, el historial puede perderse.
